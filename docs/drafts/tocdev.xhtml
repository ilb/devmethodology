ДТР

Если «Успешность работы фирмы оценивается по доходности инвестированного капитала» И «За зарабатывание и экономию денег в фирме отвечают бизнес-правила» И «Для того, чтобы обеспечить работу бизнес-правил требуется программное обеспечение», ТО «Фирме необходимо программное обеспечение для обеспечения успешной деятельности»

Если «Фирме необходимо программное обеспечение для обеспечения успешной деятельности», ТО «Фирме требуется штат сотрудников, обеспечивающих разработку и эксплуатацию ПО»

Если «ПО фирмы монолитное», ТО «Фирме требуется монолитное подразделение по разработке и эксплуатации ПО»

Если «Фирме требуется монолитное подразделение по разработке и эксплуатации ПО», ТО «При росте объемов изменений подразделение по разработке и эксплуатации ПО становится узким горлышком»

ДБР

  1. Если «Фирма оценивает и приоритизирует разработку по оценке влияния на доходность инвестированного капитала», ТО «В разработку попадают наиболее критичные задачи с точки зрения влияния на итоговые результат деятельности фирмы»

  2. «Фирма реализует требования к ПО в виде слабосвязанных сервисов»

  3. «Разработан протокол интеграции слабосвязанного ПО в цепочки бизнес-процессов»

  4. «Фирма получает возможность заказывать ПО у независимых поставщиков»

  5. «Поставщики получают возможность поставлять ПО различным потребителям, поставляя базовую часть ПО и слой интеграции»
